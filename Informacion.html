Entidad Formulario Descripcion
Articulos GetArticulos Este formulario carga automaticamente todos los datos que existen en la base de datos en la tabla ma_articulos para poder interactuar con ellos facilmente
# GetUno Esta formulario utilizando el ID que es enviado como parametro retorna los datos correspondientes a ese ID que se encuentran en la base de datos
# InsertArticulo Este formulario en el evento click del boton agregar articulo inserta en la base de datos en la tabla ma_articulos los datos ingresados por el usuario
# UpdateArticulo Al pulsar el boton editar este formulario muestra los datos seleccionados para que el usuario pueda editarlos y los actualiza en la base de datos
# DeleteArticulo Este formulario elimina de la base de datos los datos seleccionados al presionar en el boton eliminar
